📋 О платформе

CardioDecide — автономная CDSS (система клинической поддержки принятия решений) для диагностики и лечения фибрилляции предсердий. Работает полностью офлайн без интернета.

📚 Источник рекомендаций

2023 ACC/AHA/ACCP/HRS Guideline for the Diagnosis and Management of Atrial Fibrillation

Joglar JA, Chung MK et al. Circulation. 2024;149:e1–e156.

DOI: 10.1161/CIR.0000000000001193

⚙️ Встроенные алгоритмы

  • Расчёт CHA₂DS₂-VASc и HAS-BLED
  • Выбор антикоагулянта с учётом СКФ, массы тела, особых популяций
  • Контроль ЧЖС (острый и хронический)
  • Тактика кардиоверсии и ААП с учётом структурной патологии
  • Показания к катетерной аблации
  • Специальные популяции: WPW, беременность, ГКМП, ОКС, ХБП, онкология
  • Модификация факторов риска

🔄 Обмен данными (офлайн)

Для передачи данных пациента между устройствами без интернета:

Экспорт: На вкладке «Решение» нажмите «Экспорт» — получите JSON-файл или текст для передачи.

Импорт: Вставьте JSON в поле ниже и нажмите «Загрузить».

📱 Установка на телефон

iPhone/iPad: Safari → кнопка «Поделиться» → «На экран "Домой"»

Android: Chrome → меню ⋮ → «Добавить на главный экран»

После установки приложение работает без интернета.

⚠️ Отказ от ответственности

Данный инструмент предназначен для поддержки, а не замены клинического суждения врача. Все решения принимает квалифицированный специалист с учётом индивидуальных особенностей пациента.
